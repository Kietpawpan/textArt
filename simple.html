Image to Text Art

Simple Version

ASCII Image

This work is for creating a text image from a photo, inspired by Jonathan Petitcolas's tutorial page on image-to-ASCII-art converting code, at https://www.jonathan-petitcolas.com/2017/12/28/converting-image-to-ascii-art.html.

We added a simple version with the value of grayRamp of " .:-=+*#%@"; .



Modified Petitcolas's Ascii Art Converter
