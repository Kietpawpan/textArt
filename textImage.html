ASCII Image

This work relies on the work of Full-Stack Web Developer, Open-Source Contributor, Seasoned Speaker, named Jonathan Petitcolas, whose code for converting an image into ASCII art masterpiece is given at https://www.jonathan-petitcolas.com/2017/12/28/converting-image-to-ascii-art.html.

We change the output maximum size to 62 : 62, and the value of grayRamp in the simplified version to " .:-=+*#%@"; .
